4 - Calculo do Erro tipo 2, Poder do Teste e p-valor para o teste de hipteses da mdia de uma populao normal com varincia populacional conhecida

4.1 - Erro Tipo II (Teste unilateral a direita)

Considere uma amostra aleatria simples \(X_1, \ldots, X_n\) obtida de uma populao normal com mdia \(\mu\) e varincia \(\sigma^2 = 25\) conhecida.

Suponha que deseja-se testar a seguinte hiptese sobre \(\mu\), \[H_0: \mu = 500 \qquad \times \qquad H_1: \mu > 500.\]

O teste acima possui uma regio crtica definida por\[ RC = \{\bar{x}: \bar{x} > k\}.\]

Se desejamos calcular o Erro tipo II (chamado de \(\beta\)) associado ao teste acima, devemos calcular

\(\beta\) = P(Erro Tipo II) = P(No Rejeitar \(H_0\) | \(H_0\) falso) = P(\(\bar{X} \in\!\!\!\!\!/\) RC | \(H_0\) falso) = P(\(\bar{X} < k\) | \(H_0\) falso)

Para resolver esta probabilidade necessrio conhecer \(k\), mas este s conhecido se fixarmos um valor para \(\alpha\) = P(Erro Tipo I), por exemplo, vamos defini-lo como 0,05:

\(\alpha\) = P(Erro Tipo I) = P(Rejeitar \(H_0\) | \(H_0\) verdadeiro) = P(\(\bar{X} \in\) RC | \(H_0\) verdadeiro) = P(\(\bar{X} > k\) | \(\bar{X} \sim N \left(500,\frac{25}{24} \right)\)) = 0,05.

Deste modo possvel definir o valor de \(k\), basta encontramos o quantil da distribuio de \(\bar{X}\) que deixa uma rea acima dele de \(0,05\).

Definido o valor de \(k\), podemos calcular \(\beta =\) P(Erro Tipo II)?

Note que s possvel calcularmos \(\beta\) se conhecermos a distribuio de probabilidade de \(\bar{X}\). A mesma s ser conhecida se definirmos um valor para \(\mu\), ento \(\beta\) uma funo de \(\mu\), logo, podemos escrever

\(\beta(\mu)\) = P(Erro Tipo II) = P(No Rejeitar \(H_0\) | \(H_0\) falso) = P(No Rejeitar \(H_0\) | \(\mu\)) = P(\(\bar{X} \in\!\!\!\!\!/\) RC | \(\mu\)) = P(\(\bar{X} < k\) | \(\bar{X} \sim N \left(\mu,\frac{25}{24} \right)\)).

erroII.media = function(n,alfa,mu,mu0,sigma2){
  k = qnorm(1-alfa,mu0,sqrt(sigma2/n))
  prob = pnorm(k, mu, sqrt(sigma2/n))
  return(prob)
}

ggplot(data = data.frame(x = c(495,510)), aes(x = x)) + stat_function(fun = erroII.media, args = list(n = 24, alfa = 0.05, mu0 = 500, sigma2 = 25)) + xlab(expression(mu)) + ylab(expression(beta(mu)))

4.2 - Poder (Teste unilateral a direita)

poder.media = function(n,alfa,mu,mu0,sigma2){
  k = qnorm(1-alfa,mu0,sqrt(sigma2/n))
  prob = 1 - pnorm(k, mu, sqrt(sigma2/n))
  return(prob)
}

ggplot(data = data.frame(x = c(495,510)), aes(x = x)) + stat_function(fun = poder.media, args = list(n = 24, alfa = 0.05, mu0 = 500, sigma2 = 25)) + xlab(expression(mu)) + ylab(expression(1 - beta(mu)))

4.3 - P-valor (Teste unilateral a direita)

P(Observarmos valores mais extremos do que o fornecido pela amostra | \(H_0\) verdadeira).

P(\(\bar{X} > \bar{x}\) |\(H_0\) verdadeira) = P(\(\bar{X} > \bar{x}\) |\(\bar{X} \sim N \left(500,\frac{25}{24} \right)\))

pvalor.media = function(dados,mu0,sigma2){
  n = length(dados)
  x.barra = mean(dados)
  prob = 1 - pnorm(x.barra, mu0, sqrt(sigma2/n))
  return(prob)
}
