2 - Condicionamento

Em certas ocasies, queremos ou precisamos executar parte do cdigo apenas se alguma determinada condio for atendida. O R fornece algumas opes bsicas para estruturar seu cdigo: if(), if() else e ifelse(). A seguir, vamos discutir cada uma das trs opes.

2.1 - O if()

A estrutura bsica do if() a seguinte:

if(condicao){
 
  # conjunto de comandos que sero executados
  # caso a condio seja TRUE
 
}
  • Notem que, o incio do cdigo se d com o comando if seguido de parnteses e chaves;
  • Dentro do parnteses preciso conter uma condio lgica, que dever ter como resultado ou TRUE ou FALSE;
  • Dentro das chaves temos um conjunto de comandos que sero executado se a condio do parnteses for TRUE.

Veja o exemplo abaixo:

# Criando o objeto x
x = 10

 
# s executa se a condio lgica for atendida
if(x < 20){
  y = 1
}
 
# note que x foi criado
exists("x")
## [1] TRUE
x
## [1] 10
# s executa se a condio lgica for atendida
if(x > 20){
  z = 10
}

# note que z no foi criado
exists("z")
## [1] FALSE

2.2 - O if() else

Vamos ver agora como acrescentar ao if() o opcional else.

A estrutura bsica do if() else a apresentada a seguir:

if(condicao){
 
  # conjunto de comandos que sero executados
  # caso a condio seja TRUE
 
} else{
 
  # conjunto de comandos que sero executados
  # caso a condio seja FALSE
 
}
  • Percebam que o incio do cdigo se d do mesmo modo que no caso anterior, com o comando if seguido de parnteses e chaves;
  • Novamente, dentro do parnteses preciso conter uma condio lgica, que dever ter como resultado ou TRUE ou FALSE;
  • Dentro das chaves do if() temos um conjunto de comandos que sero executado se a condio do parnteses for TRUE;
  • Logo em seguida temos a presena do opcional else, seguido de chaves;
  • Dentro das chaves do else temos um conjunto de comandos que sero executado se a condio do parnteses for FALSE.
# Criando o objeto x
x = 10

 

if(x < 20){
  y = 1
  cat("O nmero y criado  igual a", y)
} else{
  y = 10
  cat("O nmero y criado  igual a", y)
}

2.2 - A funo ifelse()

Os comandos if() e if() else no so vetorizados, como podemois perceber no exemplo criado na seo anterior. O conjunto de comandos executado, s levou em considerao o primeiro elementio do vetor x e exibiu uma mensagem de warning. Uma alternativa para estes casos, utilizar a funo ifelse().

A funo ifelse() tem a seguinte estrutura bsica:

ifelse(conjunto_de_condicoes, valor_se_TRUE, valor_se_FALSE)
  • Notem que, o primeiro argumento trata-se de um conjunto de condies (que podem ser aplicadas em um vetor), que retornar vrios TRUE e FALSE;
  • J o segundo argumento trata-se do valor que ser retornado quando o elemento do conjunto_de_condicoes for TRUE;
  • E o terceiro argumento trata-se do valor que ser retornado quando o elemento do conjunto_de_condicoes for FALSE.
# Criando o objeto x
x = c(310,20,230,40)

ifelse(x >130, 1, 0)
## [1] 1 0 1 0

3 - Loopings

3.1 - O for()

Um looping que utiliza o for() no R tem a seguinte estrututa bsica:

for(j in conjunto_de_valores){
  # conjunto de comandos que 
  # sero repetidos
  # tantas vezes quanto for o tamanho do conjunto de valores
}
  • Notem que, o incio do loop se d, de forma bem similar ao if, com o comando for seguido de parnteses e chaves;
  • Dentro do parnteses, temos um indicador que ser usado durante o loop (no caso escolhemos o j para representar este indicador) e um conjunto de valores que ser iterado (chamado aqui de conjunto_de_valores);
  • Dentro das chaves, temos o conjunto de comandos que ser executado durante o loop.
x = 50
aux = 0

for(j in 1:10){

3.2 - O while()

Um looping que utiliza o while() no R tem a seguinte estrututa bsica:

while(condicao_for_satisfeita){
  # conjunto de comandos que 
  # sero repetidos
  # enquanto a condicao continuar retornando TRUE
}
  • Notem que, o incio do loop se d com o comando while seguido de parnteses e chaves;
  • Dentro do parnteses, temos uma condio que ser verificada a cada iterao;
  • Dentro das chaves, temos o conjunto de comandos que ser executado enquanto a condio retornar TRUE.
x = 1

while( x < 5){
  x = x + 1
  print(x)
}

4 - Criando funes no R

Uma das grandes vantagens de softwares estatsticos como o R o facilidade em criar funes.

Uma funo, pode ser vista, como uma sequncia de passos que voc deseja que o programa execute. Se voc faz uso, rotineiramente, de uma sequncia de passos, ao invs de ter que reescrever os passos todas as vezes, voc pode simplesmente criar uma funo.

Quando fazemos o uso de um pacote, na verdade estamos usando funes que outras pessoas j criaram.

Ao criar uma funo muito importante definir suas entradas (argumentos) e sadas (resultados a serem retornados).

Para criar uma funo no R, precisamos utilizar a funo function() que tem a seguinte estrutura bsica:

nome.funcao = function(argumentos){
  # conjunto de comandos que 
  # sero realizados
}

Vamos supor que queiramos criar uma funo que retorne a mensagem Adoro Mtodos Computacionais II toda vez que for evocada:

#Criando a funo e chamando de mensagem

mensagem = function(){
  print("Adoro Mtodos Computacionais II")
}

#evocando a funo
mensagem()
## [1] "Adoro Mtodos Computacionais II"
#Criando uma funo que soma dois nmeros

soma.minha = function(x,y){
  valor = x + y
  return(valor)
}

soma.minha(2,4)
## [1] 6

O que acontece se ao invs de somarmos dois nmeros, somarmos um nmero com uma letra?

soma.minha(2,"h")
#Melhorando a funo

soma.minha = function(x,y){
  if(class(x) != "numeric" | !is.numeric(y)){
    print("Voc deve inserir dois nmeros para que a funo possa ser executada!")
  }else{
  valor = x + y
  return(valor)    
  }
}

soma.minha(2,"h")
## [1] "Voc deve inserir dois nmeros para que a funo possa ser executada!"
soma.minha("h",2)
## [1] "Voc deve inserir dois nmeros para que a funo possa ser executada!"
soma.minha(2,4)
## [1] 6

5 - Distribuies de probabilidade

O R inclui a funcionalidade para operaes com distribuies de probabilidades. Para cada distribuio h 4 operaes bsicas indicadas pelas letras:

  • d - calcula a densidade de probabilidade de \(f(x)\) no ponto.
  • p - calcula a funo de probabilidade acumulada \(F(x)\) no ponto.
  • q - calcula o quantil correspondente a uma dada probabilidade.
  • r - sorteia uma amostra aleatria da distribuio.

Para melhor entendermos as quatro operaes, vamos considerar que \(X \sim N(25,32)\), \(Y \sim Poisson(8)\) e \(Z \sim Gama(1,5)\), isto , \[f_Z(z) = \frac{1^5}{\Gamma(5)}z^4e^{-5z}I(z > 0).\]

Suponha que desejamos obter \(P(X < 28)\).

pnorm(28,mean = 25, sd = sqrt(32))
## [1] 0.7020585

Suponha que desejamos obter uma amostra aleatria de tamanho 10 de uma normal padro.

rnorm(10)
##  [1]  0.72248551  0.04153374  0.57673451  1.75445993 -0.78136759
##  [6]  0.72074082 -0.51864815 -0.57028801  0.25308483  1.57882162

E se desejarmos obter uma amostra aleatria de \(X\) de tamanho 10?

## [1] 0.04238011

5.1 - Fazendo grficos com as distribuies de probabilidade

Plotando a funo densidade de probabilidade de uma distribuio normal.

library(ggplot2)
ggplot(data = data.frame(x = c(-10,30)), aes(x=x)) + stat_function(fun = dnorm, args = list(mean = 10, sd = 5)) + ylab("f(x)") + ggtitle("Grfico da densidade de uma N(10,25)")

Plotando a funo de distribuio acumulada de uma distribuio normal.

ggplot(data = data.frame(x = c(-10,30)), aes(x=x)) + stat_function(fun = pnorm, args = list(mean = 10, sd = 5)) + ylab("F(x)") + ggtitle("Grfico da distribuio acumulada de uma N(10,25)")
